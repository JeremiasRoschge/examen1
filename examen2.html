    Conceptos basicos de JavaScript

    Camel Case

    El termino camel case es el uso correcto de las mayusculas,
    a la hora de utiliar valores declarados por el lenguaje ejemeplo:
    document.getElementById La primera paralabra empieza con minuscula y el resto con mayusculas

    Funciones

    Hay tres formas de hacer una funcion
    Funciones de la manera normal ejemplo:
    function nombre(parametros) { funcion }

    Luego tenemos las funciones declaradas por variables: ejemplo const fun = fuction(parametros) { funcion }

    Y por ultimo las funciones flechas las cuales son las mas usadas
    const fun = (parametros)=> { funcion }

    Clases

    Las clases son aquellas que nos permiten agregarle un valor
    a un parametro y utilizar aquel valor fuera del bloque.
    Syntaxis de una clase:
    class Clase {
    constructor(parametros) {
    this.parametro = parametro
    }
    }
